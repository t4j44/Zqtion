# Zqtion Landing Page

High-performance landing page for Zqtion.com built with Next.js 14+ and cyber-minimalist design.

## Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Font:** Geist Sans

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Sticky navigation with glassmorphism blur effect
- Full-viewport hero section with animated CTAs
- Interactive iPhone demo with auto-scrolling menu
- Bento grid services section with before/after slider
- Portfolio with embedded YouTube videos
- Fully responsive design
- Dark mode cyber-minimalist aesthetic

## Design System

- **Background:** #000000 / #050505
- **Text:** #FFFFFF (headings) / #A3A3A3 (body)
- **Accent:** #0B63FF (Electric Blue)
- **Borders:** border-neutral-800

## Build

```bash
npm run build
npm start
```
